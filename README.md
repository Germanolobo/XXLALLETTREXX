# Lallettre - Alta Perfumaria

Este repositório contém o código do site institucional da **Lallettre**, combinando a sofisticação francesa com a identidade terracota da Chapada Diamantina.

## 📦 Como usar este repositório

Como os arquivos de imagem que você enviou no chat ficam protegidos e não podem ser extraídos diretamente para este arquivo compactado por questões técnicos do chat, este pacote já vem com a estrutura exata e o código pronto.

### Passo a Passo para publicar no GitHub:

1. **Descompacte** este arquivo `.zip` em uma pasta no seu computador.
2. **Baixe as 3 imagens** que você enviou aqui no chat para o seu computador.
3. **Mude o nome das imagens** exatamente para estes nomes (mantendo as maiúsculas e espaços):
   - A imagem da logo deve se chamar: `LOGO LALLETTRE 2.png`
   - A imagem do perfume 701 (borrifador) deve se chamar: `Gemini_Generated_Image_ (1).png`
   - A imagem do perfume 802 (silhueta) deve se chamar: `Gemini_Generated_Image_.png`
4. **Coloque as 3 imagens dentro da mesma pasta** onde está o arquivo `index.html`.
5. Suba todos os arquivos (`index.html`, `README.md` e as imagens) para o seu repositório no **GitHub**.

## 🚀 Como Ativar o Site (GitHub Pages)

1. No seu repositório do GitHub, vá em **Settings** (Configurações).
2. No menu lateral esquerdo, clique em **Pages**.
3. Em *Build and deployment*, mude a Source para **Deploy from a branch**.
4. Em *Branch*, selecione `main` (ou `master`) e a pasta `/ (root)`. Clique em **Save**.
5. Aguarde cerca de 1 a 2 minutos e o GitHub gerará o link oficial e público do seu site!
